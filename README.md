Full-Stack E-commerce Website built using MERN Stack (MongoDB, Express, React, Node.js). Features include User Authentication, Product Filtering, Shopping Cart, and Checkout. Level 3 Task 1 for CodSoft Internship.



# 🛒 E-Shop - Full Stack MERN E-commerce Project

This project is a fully functional E-commerce web application developed as part of the **CodSoft Internship (Level 3 Task 1)**.

### ✨ Key Features:
* **User Authentication:** Login and Registration system.
* **Product Catalog:** Browse products with names, prices, and categories.
* **Filtering:** Filter products by category (Electronics, Fashion, etc.).
* **Shopping Cart:** Add items to cart with live count and total amount calculation.
* **Mock Checkout:** Secure checkout process with payment success feedback.

### 🛠️ Tech Stack:
* **Frontend:** React.js, Axios, React Router
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Local)
* **Design:** Custom CSS for a professional look

### 🚀 Setup Instructions:
1. Clone the repo: `git clone [Your Repository Link]`
2. **Backend:** Navigate to `/backend`, run `npm install`, then `node server.js`.
3. **Frontend:** Navigate to `/frontend`, run `npm install`, then `npm start`.
